Expense Tracker-Backend
The Expense Tracker Backend is a RESTful API built using Node.js, Express.js, and MongoDB. It provides secure authentication and expense management features, allowing users to track their daily expenses efficiently.

🚀 Features
User Registration and Login
Secure JWT Authentication with HTTP-only Cookies
Forgot Password and Reset Password
Add, View, Update, and Delete Expenses
View Individual Expense Details
Monthly Expense Summary with Category-wise Breakdown
Protected Routes using Authentication Middleware
MongoDB Database Integration with Mongoose
RESTful API Architecture
